Technology

The main technology is FDM (Fuse Deposit Material), its using a heated nozzle who melt plastic and lay it down layer by layer.

Structure

Core XY or Cartesian

Material

The base filament is PLA wich is a polymer based on corn, this biodegradable filament is in theory food safe (but not the whole printer !).

It is commonly used because of its simplicity to print and it cheap cost.

Software

You need one software to translate your 3D model in a file write in printer language, these file have the extension .gcode . These software are called slicer (because they slice your model).
